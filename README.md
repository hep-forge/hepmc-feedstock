[![Build AMD64](https://github.com/hep-forge/hepmc-feedstock/actions/workflows/autoupload.amd64.yml/badge.svg)](https://github.com/hep-forge/hepmc-feedstock/actions/workflows/autoupload.amd64.yml)

Feedstock for [hepmc](https://hepmc.web.cern.ch/hepmc/) — part of [hep-forge](https://anaconda.org/hep-forge).

HepMC3 is a new rewrite of HepMC event record.

## Install

```bash
conda install -c hep-forge -c conda-forge hepmc
```
